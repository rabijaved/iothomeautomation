[IOT Home Automation]
```
*Integration for :
 DHT11/DHT22 Temperature Humidity sensor
 Motion sensor
 Soil moisture sensor
 Water pump
 Switch relays
 LED/Touch screen management
 Google home integration through IFTTT

*The project uses an Express backend and a ReactJs WebApp
```
